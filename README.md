# T-CMD — Trade Command

AI-powered crypto trading signals dashboard for BTC, ETH, SOL and trending Solana memecoins.

## Features

- 📡 **Coin Signals** — RSI, MACD, EMA crossovers, Markov Chain & Monte Carlo simulations
- 🔍 **Memecoin Scanner** — Live DexScreener integration, pump probability scoring, rug-risk detection
- 📒 **Trading Log** — Open positions, live P&L tracking, win rate stats
- 🔒 **Auth System** — Mock Passkey login, user registration, admin approval flow
- ⚙️ **Admin Panel** — User management, feature access control per user

## APIs Used (all free, no keys required)

- [CoinGecko](https://www.coingecko.com/en/api) — prices, OHLCV data
- [DexScreener](https://docs.dexscreener.com/) — memecoin scanning
- [Alternative.me](https://alternative.me/crypto/fear-and-greed-index/) — Fear & Greed index
